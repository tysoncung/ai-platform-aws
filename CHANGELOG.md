# Changelog

## 0.1.0 (2026-02-27)

### Features
- AI Gateway with 6 providers (Bedrock, OpenAI, Anthropic, Azure OpenAI, Google Gemini, Cohere)
- TypeScript SDK with streaming support
- RAG pipeline with MongoDB Atlas Vector Search
- Agent framework with ReAct loop, tools, guardrails, and human-in-the-loop
- React admin dashboard (5 pages)
- OpenAPI 3.1 contract-first design
- BYOK (Bring Your Own Key) multi-tenant support
- Response caching and rate limiting
- OpenTelemetry tracing, Pino logging, Prometheus metrics
- 112 tests across all packages
- 7 example applications
- AWS CDK infrastructure (ECS + Lambda)
- Docker Compose for local development
