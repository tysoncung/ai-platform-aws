# AcmeCloud Platform Overview

AcmeCloud is a next-generation cloud platform designed for AI-native applications.

## Key Features

- **Auto-scaling inference**: Automatically scales GPU instances based on request volume
- **Model registry**: Version and deploy ML models with one command
- **Vector storage**: Built-in vector database for embeddings (up to 10M vectors per index)
- **Edge deployment**: Deploy models to 200+ edge locations worldwide

## Pricing

- Starter: $99/month - 1M API calls, 10GB vector storage
- Pro: $499/month - 10M API calls, 100GB vector storage, priority support
- Enterprise: Custom pricing - unlimited calls, dedicated infrastructure

## Supported Models

AcmeCloud supports all major model providers through its unified API gateway:
- AWS Bedrock models
- OpenAI models (GPT-4o, GPT-4o-mini)
- Open-source models via SageMaker endpoints
